Preguntas

¿Qué es un DNS?

El DNS traduce los nombres de dominio a direcciones IP para que los navegadores puedan cargar los recursos de Internet. Cada dispositivo conectado a Internet tiene una dirección IP única que otros equipos pueden usar para encontrarlo.

¿Qué es un SSL

Son credenciales para el entorno online; se crean de manera exclusiva para un dominio y un servidor web determinados, y los autentica el proveedor de certificados SSL. Cuando un navegador se conecta con un servidor, este último envía la información de identificación al navegador.
